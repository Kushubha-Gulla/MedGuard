# MedGuard

MedGuard is a student project designed to help users verify medicines with simple scanning and information features.  
This project is built with **HTML, CSS, and JavaScript** (future scope: MERN stack).

## Features
- 🌐 Language selection (English, Hindi, Telugu)
- 📷 Upload or camera option for medicine photos
- 🧾 Placeholder for medication details, expiry, and side effects
- 🔊 Voice output for accessibility
- 📞 Emergency contact link (108)

## Tech Stack
- HTML (structure)
- CSS (styling)
- JavaScript (interactivity)
- MERN stack (future integration)

## Disclaimer
⚠️ This app is a student project for educational purposes.  
Information may not be accurate. Always consult a licensed doctor before taking any medication.
